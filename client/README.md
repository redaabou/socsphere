# SOCSphere IOC Lookup

## Overview

SOCSphere IOC Lookup is a threat-intelligence
enrichment tool for IP addresses, domains, and
file hashes.

## Features

- Automatic IOC type detection
- VirusTotal enrichment
- AbuseIPDB IP enrichment
- Transparent risk calculation
- Responsive interface
- Secure backend API proxy

## Technologies

- React
- Tailwind CSS
- Node.js
- Express
- VirusTotal API
- AbuseIPDB API

## Security Considerations

- API keys are stored server-side
- Request rate limiting
- Input validation
- HTTP security headers
- Limited request body size

## Disclaimer

This project is intended for educational and
portfolio use. Reputation results should be reviewed
alongside additional context before making security
decisions.
